# Systemy-Analityczno-Raportowe-projekt
Projekt z przedmiotu Systemy Analityczno-Raportowe wykonany w zespole ośmioosobowym na piątym semestrze studiów na WAT 

Projekt obejmował trzy główne etapy: 
- Hurtownia danych z wykorzystaniem modelu gwiazdy i płatka śniegu zrealizowana w Supabase (PostGreSQL)
- Raporty i wizualizacje na podstawie danych z hurtowni, realizacja w Power BI
- Aplikacja do wprowadzania danych do bazy, realizacja: Formularz desktopowy w Python + PySide6
